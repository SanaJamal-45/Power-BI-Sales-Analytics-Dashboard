# Power-BI-Sales-Analytics-Dashboard


## Project Overview

This project is a comprehensive **Sales Analytics Dashboard** developed using **Power BI** to analyze business sales performance across multiple dimensions including revenue, regional sales, customer segments, product categories, and shipping operations.

The dashboard transforms raw sales data into interactive visual reports that help businesses make informed decisions through data-driven insights.

# Business Problem

Organizations generate large volumes of sales data, making it difficult to manually identify trends and performance issues. This dashboard helps stakeholders:

- Monitor business growth
- Compare regional performance
- Analyze customer segments
- Improve operational efficiency
- Identify profitable product categories
- Make strategic business decisions

---

# Dashboard Pages

### 1. Executive Summary
Provides a high-level overview of sales, profit, and business KPIs.

### 2. Regional Performance
Analyzes sales performance across different geographical regions.

### 3. Segment Performance
Shows contribution and performance of different customer segments.

### 4. Shipping & Operations
Tracks shipping efficiency, operational trends, and logistics performance.

### 5. Sub Category Analysis
Provides detailed insights into product sub-categories and profitability.
